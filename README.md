# r2-d2-
Transform Arduino into R2-D2! Authentic beeps &amp; boops, nervous head shakes, smart obstacle avoidance. Control via smartphone Bluetooth or explore autonomously. Follows lines, scans with servo head, makes contextual sounds. Built with Arduino Uno, HC-05, HC-SR04 sensor, dual IR. Beginner-friendly, 4-6hr build, $40-60. Star Wars robotics fun!
